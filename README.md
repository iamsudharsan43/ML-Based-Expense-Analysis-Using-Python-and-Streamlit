# 💰 ML-Based Expense Analysis Using Python & Streamlit

A **machine learning–assisted expense tracking and analysis application** built using **Python** and **Streamlit**.
This project helps users track expenses, visualize spending patterns, and gain insights into their financial behavior through interactive dashboards.

---

## 🚀 Features

* 📊 Add, view, and manage daily expenses
* 🗂️ Category-wise expense classification
* 📈 Interactive **bar charts & pie charts** for spending analysis
* 📅 Date-based and month-wise filtering
* 🤖 Basic ML-driven insights (pattern analysis / trends)
* 🧾 Clean and simple UI built with **Streamlit**
* 🔐 User-friendly and lightweight setup

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/iamsudharsan43/ML-Based-Expense-Analysis-Using-Python-and-Streamlit.git
cd ML-Based-Expense-Analysis-Using-Python-and-Streamlit
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
```

Activate:

* **Windows**

```bash
venv\Scripts\activate
```

* **Mac/Linux**

```bash
source venv/bin/activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
streamlit run app.py
```
